TRIGONOMETRY

PROBLEMS INVOLVONG CLIMATE CHANGE

PROBLEM 1

An iceberg from Antarctica is reportedly melting because of an increase in temperature from the weather. According to the data, the specific iceberg was measured as 20 meters wide where it creates the angle of elevation from the top to its foot on a horizontal plane of 40°. If the iceberg sank resulting in a decrease of 5 degrees for the angle, how tall is the iceberg now?

Hint:
Uh oh! The horizontal plane of the iceberg sank 5 DEGREES.
Remember that 20m wide is the width of the whole iceberg. Also, find the other 2 sides in order to
know which trigonometric function to use

SOLUTION

tan = opposite/adjacent Because the adjacent side is given, we'll apply the tangent function.

tan(35º) = x/10 Substitute the given values (As said in the problem, the iceberg sank resulting in a decrease of 5 degrees for the angle that’s why the angle is 35º.)

10tan(35º) = x Multiply both sides by 10

x = 10tan(35º) Swap the sides of the equation

x = 10 x 0.7 Find the value of tan(35º)

x = 7m

Answer: The iceberg is 7 meters tall if the iceberg sank resulting in a decrease of 5 degrees for the angle.

PROBLEM 2

Based on the news of Weather Daily, there is an increase in the intensity of cyclones because of issues regarding climate change. As they observed, the cyclone from the Philippines already moved somewhere in South China. If the distance between Cambodia and Philippines is 2,410 km, what is the computed distance travelled by the cyclone if it has an inclination angle of 40 degrees?

Hint:
The distance traveled by the cyclone from the Philippines to South China acts as the hypotenuse of the imagined right triangle. Don't forget to find the other 2 sides in order to know which trigonometric function to use.

SOLUTION

cos = adjacent/hypotenuse Because the adjacent side is given, we'll apply the cosine function.

cos(40º) = 2,410/x Substitute the given values

cos(40º)x = 2, 410 Multiply both sides by x

x = 2,410/cos(40º) Divide both sides by cos(40º)

x = 2,410 x 1cos/(40º)

x = 2,410 x sec(40º) Use the trigonometric identities, substitute 1/cosθ to secθ.

x = 2,410 x 1.30540728-

Find the value of sec(40º)

x = 3146.0315km

Answer: The computed distance travelled by the cyclone is roughly 3, 146.0315 km if it has an inclination angle of 40 degrees.

PROBLEM 3

NASA launched an Earth-orbiting satellite to get more information about Climate Change. Because of its bird’s eye view, experts collected data with the help of the machine which is 35,800 km above our planet. Since the station is found somewhere in Africa, how far is the satellite if it creates a 70-degree angle?

Hint:
Find the other 2 sides in order to know which trigonometric function to use. Once known that the distance between the satellite and Africa is the imagined hypotenuse of the triangle, since the opposite of the angle was given, which trigonometric function should we use now?

SOLUTION

sin=opposite/hypotenuse Because the opposite side is given, we'll apply the sine function.

sin(70º) = 35,800/x Substitute the given values

sin(70º)x = 35,800 Multiply both sides by x

x = 35,800/sin(70º) Divide both sides by sin(70º)

x = 35,800 1/sin(70º)

x = 35,800 x csc(70º) Use the trigonometric identities, substitute 1/sinθ to cscθ.

x = 2,410 x 1.0641777 Find the value of csc(70º)

x = 38097.56km

Answer: The satellite is estimated to be 38,097.56 km far from the station that is found in Africa if it creates a 70-degree angle.
